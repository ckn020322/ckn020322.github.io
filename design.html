Typography System

Clear, readable fonts with ADHD-friendly spacing and hierarchy:

Main Heading - Inter 800

사용 예시: 페이지 제목 및 주요 섹션

Font size: 2.8rem • Line height: 1.2 • Color: Primary Orange

Section Heading - Inter 700

사용 예시: 섹션 헤더 및 콘텐츠 구분

Font size: 2rem • Orange accent underline • Clear hierarchy

Card Title - Inter 600

사용 예시: 카드 제목 및 서브헤더

Font size: 1.4rem • Good visual weight for scanning

Body Text - Open Sans Regular

한국어 텍스트: 노토 산스 KR 사용

Line height: 1.6 for comfortable reading • ADHD-friendly spacing between paragraphs
